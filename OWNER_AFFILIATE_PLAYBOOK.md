# ihelpwithai.com affiliate playbook

Last updated: March 25, 2026

This site is now wired so the public CTA automatically switches from `officialUrl` to `affiliateUrl` whenever you add your unique partner link inside [`tools-starter.json`](/Users/jorgesanchez/Desktop/IHWAI/tools-starter.json). The site also adds the right `rel="sponsored"` treatment when an affiliate link is present.

## Best first programs to apply for

1. Grammarly
Official page: https://www.grammarly.com/affiliates
Why start here: public affiliate page, strong mainstream fit, easy tool to recommend to a broad audience.

2. Descript
Official page: https://www.descript.com/affiliate
Why start here: creator-friendly offer and clear application flow.

3. ElevenLabs
Official page: https://elevenlabs.io/affiliates
Why start here: clear public affiliate page and a strong fit for voice, creator, and business content pages.

4. Fireflies.ai
Official page: https://fireflies.ai/partnership/affiliate
Why start here: meeting notes are easy to explain and the site already has business-use visitors.

5. Murf
Official page: https://murf.ai/partner-with-us/affiliate
Why start here: straightforward recurring commissions and an easy voiceover use case.

6. Riverside
Official page: https://riverside.fm/affiliate-program
Why start here: strong creator and podcast audience fit.

7. Zapier
Official page: https://zapier.com/l/partners
Why start here: not a simple consumer affiliate, but the creator/partner path is valuable if you plan to publish automation content.

8. Writesonic
Official page: https://docs.writesonic.com/docs/writesonic-affiliate-program
Why start here: good fit if you want SEO and marketing tool coverage.

## Programs worth watching

- Notion affiliate program
Official page: https://www.notion.com/de/affiliates
Status: official page says the affiliate program is not currently accepting new partners.

- Notion solutions partner program
Official page: https://www.notion.com/help/become-a-notion-partner
Status: useful if you later want consulting or implementation revenue around Notion, not just referral income.

## After you get approved

1. Open [`tools-starter.json`](/Users/jorgesanchez/Desktop/IHWAI/tools-starter.json).
2. Add your unique link to the `affiliateUrl` field for each approved tool.
3. Run `npm run build` in [`/Users/jorgesanchez/Desktop/IHWAI`](/Users/jorgesanchez/Desktop/IHWAI).
4. Redeploy the [`public`](/Users/jorgesanchez/Desktop/IHWAI/public) folder to Netlify.
